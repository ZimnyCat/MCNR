# MCNR
Оригинальные плагины сервера mcnr.xyz

### MCNR-Door
Плагин для сервера аутентификации. Команда `/join` для перехода на основной сервер

### MCNR-Proxy
Плагин для прокси-сервера. Показывает случайное MOTD (описание сервера) в списке серверов, обрабатывает подключения

### MCNR-Utils
Плагин для основного сервера. Автоперезапуск, сообщения в табе, система личных сообщений, игнор игроков, баг-репорты и некоторые другие вспомогательные фичи
